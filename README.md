## 📝 What is CForum?

CForum is a user-friendly forum application. It does not require you to manage a server. You can run it directly on your computer and use it through your browser. CForum offers:

- Posting messages and replies
- Nested comment threads for clear discussions
- Uploading images in posts and comments
- Two-factor authentication (2FA) for added security
- Fast access through smart edge routing technology

This software runs on Windows with no coding needed.

## 💻 System Requirements

Before installing, make sure your computer meets these basic requirements:

- Operating System: Windows 10 or newer
- RAM: 4 GB or more
- Free Disk Space: At least 200 MB
- Internet Connection: Required for setup and some features
- Browser: Latest version of Chrome, Edge, or Firefox is recommended

No additional software or servers are required.

## 🔐 Setting Up Your Account

When you open CForum for the first time, create an account by following these steps:

- Enter a username and password.
- Choose to enable two-factor authentication (2FA) for better security. You will need an authenticator app on your phone (like Google Authenticator) for this.
- Confirm your email address if prompted. This is optional but recommended.

Your account lets you post messages, reply to others, and upload images.

## 📂 Using CForum Features

Here’s how to use the main features:

### Posting and Commenting

- Click `New Post` to start a discussion.
- Write your message in the text box.
- Use the toolbar to add images or format text.
- Hit `Publish` to share it with others.
- Reply to posts by clicking `Reply` under any message.
- Reply threads are nested to keep conversations organized.

### Uploading Images

- Click the upload image button to add pictures to your posts or replies.
- Supported formats include JPG, PNG, and GIF.
- Image size limit per file is 5 MB.

### Account Security

- Access your profile settings to turn on or off two-factor authentication.
- Change your password anytime from the settings page.
- Log out when you finish for safety.

## 🛠 Troubleshooting Tips

- If the installer does not start, check your antivirus or Windows Defender. It might block unknown software.
- If CForum does not open after installation, restart your computer and try again.
- Use the default web browser if you have problems with others.
- Update your browser to the latest version.
- For slow loading, check your internet connection.
